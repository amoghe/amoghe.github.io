Fix backlight control keys on a Lenovo T440s

Without this, the hardware keys for brightness control worked at a very coarse granularity (only 4 levels seemed to work). The solution I found here worked for me.

TL;DR: Simply add this to the GRUB_CMDLINE_LINUX_DEFAULT line in /etc/default/grub: GRUB_CMDLINE_LINUX_DEFAULT="acpi_backlight=vendor quiet splash" and run sudo update-grub. Reboot and you will be able to change the backlight brightness in finer steps.
